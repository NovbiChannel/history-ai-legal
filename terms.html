Раздел 02

Описание сервиса

«История ИИ» — мобильное приложение для изучения истории и решения задач с помощью искусственного интеллекта. Основные функции:

  • Сканирование задач через камеру устройства или выбор фото из галереи.
  • Обрезка изображения для точной передачи условия задачи на сервер.
  • Чат с ИИ-ассистентом: текстовые вопросы и задачи по фото.
  • История чатов с возможностью возврата к прошлым решениям.
  • Исторические даты в разделе «Инструменты» — события по эпохам с датами и описаниями, поиск по дате или названию события, работает локально без интернета.
  • Светлая и тёмная темы оформления.

Все запросы обрабатываются через собственный сервер Приложения. Ответы ИИ носят образовательно-справочный характер и не заменяют консультацию квалифицированного преподавателя.
